Kripto Jaunumi is a web application designed as a learning project to explore and understand the fundamentals of ReactJS while building a platform for crypto market news and updates. The goal of this project was to develop a user-friendly and interactive website that leverages ReactJS, along with the API integration using Axios, to fetch real-time data and display information about cryptocurrencies. Through this learning project, the i gained valuable experience in working with ReactJS and understanding its core concepts, including components, functions, and state management. By integrating Axios for API communication, the i learned how to retrieve and display real-time data from the crypto market.
