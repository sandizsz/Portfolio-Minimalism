In 2021, during my internship at Piere, I worked on the SparkleHeart WordPress theme development. This project was a collaborative endeavor where I partnered with a skilled designer to create a responsive and mobile-friendly theme. The designer used Figma to draft an engaging and intuitive user interface. My task was to breathe life into this design, transforming it into a dynamic WordPress theme using a blend of WordPress, CSS, HTML, JavaScript, Advanced Custom Fields (ACF), and Custom Post Types (CPT). The SparkleHeart theme is not only visually appealing but also responsive and mobile-friendly, ensuring a seamless experience across various devices and screen sizes. HTML and CSS were utilized to structure and style the web pages, and JavaScript was used to introduce dynamic functionality, guaranteeing a smooth user experience. The integration of ACF and CPT with WordPress further extended the flexibility and customization of content, catering to diverse user requirements. This experience at Piere allowed me to hone my web development skills, demonstrating my ability to collaboratively translate design blueprints into functional, user-centric solutions.
